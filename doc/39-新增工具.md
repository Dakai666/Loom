# 新增工具

本指南說明如何在 Loom 中註冊新工具。

---

## 工具結構

```python
# loom/core/harness/tool.py
@dataclass
class Tool:
    """工具定義"""
    
    name: str                      # 工具名稱
    description: str               # 描述
    trust_level: str              # 信任級別（SAFE/GUARDED/CRITICAL）
    
    async def execute(self, args: dict, context: dict) -> ToolResult:
        """執行工具"""
        pass
    
    def get_openai_schema(self) -> dict:
        """返回 OpenAI 格式的 schema"""
        pass
```

---

## 步驟 1：創建工具類

```python
# loom/tools/my_tool.py
from dataclasses import dataclass
from loom.core.harness.tool import Tool, ToolResult

@dataclass
class MyTool(Tool):
    """我的自訂工具"""
    
    name = "my_tool"
    description = "這是一個自訂工具的描述"
    trust_level = "SAFE"
    
    async def execute(self, args: dict, context: dict) -> ToolResult:
        """執行工具"""
        
        # 取得參數
        param1 = args.get("param1")
        param2 = args.get("param2", "default_value")
        
        try:
            # 工具邏輯
            result = self._do_something(param1, param2)
            
            return ToolResult(
                success=True,
                output=result,
                metadata={"param1": param1}
            )
        
        except Exception as e:
            return ToolResult(
                success=False,
                error=str(e)
            )
    
    def get_openai_schema(self) -> dict:
        """返回 OpenAI schema"""
        return {
            "type": "function",
            "function": {
                "name": self.name,
                "description": self.description,
                "parameters": {
                    "type": "object",
                    "properties": {
                        "param1": {
                            "type": "string",
                            "description": "第一個參數"
                        },
                        "param2": {
                            "type": "string",
                            "description": "第二個參數（可選）",
                            "default": "default_value"
                        }
                    },
                    "required": ["param1"]
                }
            }
        }
    
    def _do_something(self, param1: str, param2: str) -> str:
        """實際的工具邏輯"""
        return f"Result: {param1}, {param2}"
```

---

## 步驟 2：註冊工具

### 方式 1：程式化註冊

```python
from loom.core.harness.registry import ToolRegistry

# 創建工具
my_tool = MyTool()

# 註冊
registry = ToolRegistry.get_instance()
registry.register(my_tool)
```

### 方式 2：loom.toml 配置

```toml
[tools]

# 內建模具已自動註冊

# 自訂工具
[[tools.custom]]

[[tools.custom.my_tool]]
name = "my_tool"
description = "我的自訂工具"
trust_level = "SAFE"
module = "loom.tools.my_tool"  # Python 模組路徑
```

### 方式 3：Plugin 註冊

```python
# 在 Plugin 中註冊
class MyPlugin(LoomPlugin):
    async def install(self, registry: PluginRegistry):
        registry.register_tool(MyTool())
```

## 步驟 3：驗證工具

```bash
# 驗證工具定義
loom tool validate --tool my_tool

# 測試工具
loom tool test my_tool param1="hello" param2="world"
```

## ToolResult 結構

```python
@dataclass
class ToolResult:
    """工具執行結果"""
    
    success: bool                  # 是否成功
    output: Any = None           # 輸出結果
    error: str | None = None     # 錯誤訊息
    metadata: dict = field(default_factory=dict)  # 額外資料
```

---

## Trust Level

| 級別 | 說明 | 範例 |
|------|------|------|
| SAFE | 純讀取，不修改任何內容 | read_file, web_search |
| GUARDED | 修改本地檔案或有限副作用 | write_file, run_bash |
| CRITICAL | 刪除、遠端操作等高風險 | rm, ssh, sql |

## 完整範例

```python
# loom/tools/calculator.py
from dataclasses import dataclass
from loom.core.harness.tool import Tool, ToolResult

@dataclass
class CalculatorTool(Tool):
    """計算機工具"""
    
    name = "calculator"
    description = "執行數學計算"
    trust_level = "SAFE"
    
    async def execute(self, args: dict, context: dict) -> ToolResult:
        expression = args.get("expression")
        
        try:
            # 安全評估數學表達式
            import ast
            import operator
            
            result = self._safe_eval(expression)
            
            return ToolResult(
                success=True,
                output=str(result),
                metadata={"expression": expression}
            )
        
        except Exception as e:
            return ToolResult(
                success=False,
                error=f"計算錯誤: {str(e)}"
            )
    
    def _safe_eval(self, expression: str) -> float:
        """安全評估數學表達式"""
        allowed_ops = {
            ast.Add: operator.add,
            ast.Sub: operator.sub,
            ast.Mult: operator.mul,
            ast.Div: operator.truediv,
            ast.Pow: operator.pow,
        }
        
        def eval_node(node):
            if isinstance(node, ast.Num):
                return node.n
            elif isinstance(node, ast.BinOp):
                left = eval_node(node.left)
                right = eval_node(node.right)
                return allowed_ops[type(node.op)](left, right)
            else:
                raise ValueError(f"不支援的運算: {ast.dump(node)}")
        
        tree = ast.parse(expression, mode="eval")
        return eval_node(tree.body)
    
    def get_openai_schema(self) -> dict:
        return {
            "type": "function",
            "function": {
                "name": self.name,
                "description": self.description,
                "parameters": {
                    "type": "object",
                    "properties": {
                        "expression": {
                            "type": "string",
                            "description": "數學表達式，如 '2 + 3 * 4'"
                        }
                    },
                    "required": ["expression"]
                }
            }
        }
```

## 總結

新增工具的步驟：

1. 創建 Tool 子類
2. 實現 `execute()` 和 `get_openai_schema()`
3. 選擇 Trust Level
4. 註冊到 ToolRegistry
5. 驗證和測試
