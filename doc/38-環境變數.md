# 環境變數

Loom 支援環境變數設定，可在 `.env` 檔案或系統環境中設定。

---

## .env 檔案

在 Loom 的資料目錄或專案根目錄建立 `.env` 檔案：

```bash
# ~/.loom/.env  或  ./project/.env

# API Keys
MINIMAX_API_KEY=your_minimax_api_key
OPENAI_API_KEY=your_openai_api_key

# Webhook Tokens
WEBHOOK_TOKEN=your_webhook_token
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...

# Telegram
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_chat_id

# GitHub
GITHUB_TOKEN=your_github_token

# Brave Search
BRAVE_SEARCH_API_KEY=your_brave_api_key

# Email
EMAIL_USER=your_email@example.com
EMAIL_PASSWORD=your_email_password

# Database (可選)
DATABASE_URL=sqlite:///~/.loom/data/loom.db
```

---

## 支援的環境變數

### API Keys

| 變數 | 說明 | 必要 |
|------|------|------|
| `MINIMAX_API_KEY` | MiniMax API Key | 是 |
| `OPENAI_API_KEY` | OpenAI API Key | 是 |
| `ANTHROPIC_API_KEY` | Anthropic API Key | 用於 claude-* 模型 |
| `AZURE_OPENAI_API_KEY` | Azure OpenAI API Key | 否 |
| `AZURE_OPENAI_ENDPOINT` | Azure OpenAI Endpoint | 否 |

### Discord Bot（v0.2.3.1 + v0.2.5.2）

| 變數 | 說明 |
|------|------|
| `DISCORD_BOT_TOKEN` | Discord Bot 的 bot token（`loom discord start` 必填）|
| `DISCORD_WEBHOOK_URL` | Discord Webhook URL（Notifier 適配器用）|
| `DISCORD_CHANNEL_ID` | 可選，限制 Bot 只能操作指定频道 |
| `DISCORD_USER_ID` | 可選，限制 Bot 只能回覆指定用戶 |

> `DISCORD_BOT_TOKEN` 僅用於 `loom discord start`。普通的 Discord Webhook 通知（`DiscordNotifier`）只需要 `DISCORD_WEBHOOK_URL`。

### Notification

| 變數 | 說明 |
|------|------|
| `TELEGRAM_BOT_TOKEN` | Telegram Bot Token |
| `TELEGRAM_CHAT_ID` | Telegram Chat ID |
| `DISCORD_WEBHOOK_URL` | Discord Webhook URL |
| `WEBHOOK_TOKEN` | Generic Webhook Token |

### Search

| 變數 | 說明 |
|------|------|
| `BRAVE_SEARCH_API_KEY` | Brave Search API Key |

### GitHub

| 變數 | 說明 |
|------|------|
| `GITHUB_TOKEN` | GitHub Personal Access Token |

### Database

| 變數 | 說明 | 預設 |
|------|------|------|
| `DATABASE_URL` | 資料庫連接字串 | `sqlite://~/.loom/data/loom.db` |

### Logging

| 變數 | 說明 | 預設 |
|------|------|------|
| `LOG_LEVEL` | 日誌級別 | `INFO` |
| `LOG_FILE` | 日誌檔案路徑 | `~/.loom/logs/loom.log` |

### 其他

| 變數 | 說明 |
|------|------|
| `LOOM_CONFIG_PATH` | 設定檔路徑 |
| `LOOM_DATA_DIR` | 資料目錄 |
| `LOOM_PLUGINS_DIR` | Plugins 目錄 |

## 載入順序

環境變數的載入順序（後面覆蓋前面）：

```
1. 系統環境變數
       ↓
2. ~/.loom/.env
       ↓
3. ./project/.env
       ↓
4. 命令列 --env 指定的檔案
```

---

## Discord Bot Token 取得方式

1. [Discord Developer Portal](https://discord.com/developers/applications) 建立 Application
2. 建立 Bot 用戶
3. 在 OAuth2 → URL Generator 中選擇 `bot` + `applications.commands` scopes
4. 使用生成的 URL 將 Bot 加入你的 Server
5. 複製 Bot Token 並設定 `DISCORD_BEBOT_TOKEN`

## 驗證環境變數

```bash
# 查看所有已設定的環境變數（不含敏感值）
loom env list

# 檢查必要的環境變數是否設定
loom env validate

# 輸出範例
$ loom env validate

Checking required environment variables...

✓ MINIMAX_API_KEY is set
✓ OPENAI_API_KEY is set
✗ TELEGRAM_BOT_TOKEN is not set (optional)
✗ DISCORD_WEBHOOK_URL is not set (optional)

Result: Ready to run (2/2 required variables set)
```

## 安全考量

### 敏感資訊

`.env` 檔案包含敏感資訊，**不要** commit 到版本控制：

```bash
# .gitignore
.env
.env.*
*.env
```

### 環境變數過濾

Loom CLI 在輸出日誌時會自動過濾敏感環境變數：

```
# 輸出會變成
MINIMAX_API_KEY=****（已遮罩）
DISCORD_BOT_TOKEN=****（已遮罩）
```

## 範例 .env 檔案

```bash
# ═══════════════════════════════════════════════════════════
# Loom Environment Variables
# ═══════════════════════════════════════════════════════════

# ─── API Keys ───────────────────────────────────────────────
MINIMAX_API_KEY=your_minimax_api_key_here
OPENAI_API_KEY=your_openai_api_key_here
ANTHROPIC_API_KEY=your_anthropic_api_key_here

# ─── Discord Bot ────────────────────────────────────────────
DISCORD_BOT_TOKEN=your_discord_bot_token_here
DISCORD_CHANNEL_ID=123456789               # optional
DISCORD_USER_ID=987654321                 # optional
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/xxx/yyy

# ─── Notifications ──────────────────────────────────────────
WEBHOOK_TOKEN=your_webhook_token_here

# ─── Search ─────────────────────────────────────────────────
BRAVE_SEARCH_API_KEY=your_brave_api_key

# ─── GitHub ─────────────────────────────────────────────────
GITHUB_TOKEN=your_github_token_here

# ─── Logging ────────────────────────────────────────────────
LOG_LEVEL=INFO
```

## 總結

| 類別 | 變數 |
|------|------|
| API Keys | `MINIMAX_API_KEY`, `OPENAI_API_KEY`, `ANTHROPIC_API_KEY` |
| Discord | `DISCORD_BOT_TOKEN`, `DISCORD_WEBHOOK_URL`, `DISCORD_CHANNEL_ID`, `DISCORD_USER_ID` |
| Notifications | `TELEGRAM_*`, `WEBHOOK_TOKEN` |
| Search | `BRAVE_SEARCH_API_KEY` |
| GitHub | `GITHUB_TOKEN` |
| Database | `DATABASE_URL` |
| Logging | `LOG_LEVEL`, `LOG_FILE` |
