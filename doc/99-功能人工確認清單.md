## 2. Harness Layer（工具生命週期管理）

### 2.1 Middleware Pipeline
- [ ] **Pipeline 順序正確**：工具呼叫依序經過所有已註冊的 middleware
- [ ] **Middleware 不阻斷流程**：正常工具執行不會被 middleware  中斷
- [ ] **Middleware 能讀取或修改上下文**：如 LogMiddleware 正確寫入軌跡

### 2.2 Trust Level
- [ ] **SAFE 工具直接執行**：不需確認，直接回傳結果
- [ ] **GUARDED 工具觸發警告**：有日誌或提示，但不放行
- [ ] **CRITICAL 工具等待確認**：明確暫停，等待 human approval
- [ ] **未識別的 trust level 有合理降級**：預設為 CRITICAL 或拋出警告

### 2.3 Tool Result 封裝
- [ ] **工具失敗不拋异常**：錯誤封裝在 ToolResult.success=False
- [ ] **錯誤訊息有結構**：ToolResult 包含可解析的 error 欄位
- [ ] **agent 能根據 ToolResult 條件分支**：不會因为错误结果而 crash
