## skills/ 詳解（用戶技能包，v0.2.6.1）

> **命名背景**：`extensibility/` 根目錄（用戶-authored 技能包）與 `loom/extensibility/`（框架代碼）同名易混淆。v0.2.6.1 起前者改名為 `skills/`。

```
skills/                          # 用戶自訂技能包（由 Loom 自動發現）
└── <package-name>/
    ├── __init__.py             # 可選，Package 入口
    ├── loom_tools.py           # 用 @loom.tool 裝飾的工具（自動發現）
    ├── SKILL.md               # 技能描述 Markdown（被 Skill Import Pipeline 解析）
    └── manifest.toml           # 可選，額外元數據
```

**與 Plugin 的分工：**

| 放置位置 | 類型 | 自動發現 |
|----------|------|---------|
| `~/.loom/plugins/<name>/` | Plugin（含 tools/middleware/notifiers）| ✅ |
| `./skills/<name>/` | Skill Package（技能 Markdown）| ✅（Skill Import） |
| `loom/extensibility/` | 框架內建模組 | ✅（LensRegistry） |
