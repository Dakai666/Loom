# 目錄結構

## 專案根目錄

```
Project_Next/
├── loom.toml               # 主設定檔（triggers / cognition / memory / notify）
├── pyproject.toml          # Python 套件定義（loom[api] 等 extras）
├── SOUL.md                 # 核心身份檔（永駐 system prompt）
├── Agent.md                # 專案身份檔（Phase 4，project-scoped）
├── personalities/         # 人格 markdown 目錄
│   ├── adversarial.md
│   ├── architect.md
│   ├── minimalist.md
│   ├── operator.md
│   ├── researcher.md
│   └── barista.md          # 新增：咖啡師人格
├── loom/                   # 主要程式碼
│   ├── __init__.py
│   ├── core/               # 核心層
│   │   ├── harness/        # Harness Layer
│   │   ├── memory/         # Memory Layer
│   │   ├── cognition/      # Cognition Layer
│   │   └── tasks/          # Task Engine
│   ├── autonomy/           # Autonomy Engine
│   ├── notify/             # Notification Layer
│   │   └── adapters/       # Notifier 適配器
│   ├── extensibility/      # Extensibility Layer
│   │   ├── lens.py         # BaseLens 抽象
│   │   ├── hermes.py       # HermesLens
│   │   ├── claw.py         # OpenAIToolsLens（原 ClawCodeLens）
│   │   ├── pipeline.py     # Skill Import Pipeline
│   │   └── adapter.py      # AdapterRegistry + @loom.tool
│   ├── agent/              # Sub-agent（Phase 5E）
│   ├── platform/          # Platform Layer
│   │   └── cli/           # CLI 實作
│   └── platform/          # REST API（Phase 5 REST API）
│       └── api/
├── doc/                    # 文檔目錄（本目錄）
├── diary/                  # 日記（開發記錄）
├── scripts/                # 輔助腳本
└── tests/                  # 測試套件（352 tests）
```

## loom/core/ 詳解

### loom/core/harness/（工具生命週期管理）

| 檔案 | 類別 / 函數 | 說明 |
|------|-------------|------|
| `middleware.py` | `Middleware`, `ToolCall`, `ToolResult` | Middleware 抽象基類 + Pipeline |
| | `LogMiddleware` | 終端 Rich 格式化輸出 |
| | `TraceMiddleware` | 計時 + EpisodicMemory 寫入 |
| | `BlastRadiusMiddleware` | Trust Level 確認攔截 |
| `permissions.py` | `TrustLevel`, `PermissionContext` | 三級權限 + session 內預授權 |
| `registry.py` | `ToolDefinition`, `ToolRegistry` | 工具定義 + 雙 schema 輸出 |

### loom/core/memory/（記憶系統）

| 檔案 | 類別 / 函數 | 說明 |
|------|-------------|------|
| `store.py` | `SQLiteStore` | SQLite WAL 後端，統一初始化 |
| `episodic.py` | `EpisodicEntry`, `EpisodicMemory` | 順序事件寫入，session-scoped |
| `semantic.py` | `SemanticEntry`, `SemanticMemory` | Key-value 長期事實，含向量支援 |
| `procedural.py` | `SkillGenome`, `ProceduralMemory` | EMA confidence 技能管理 |
| `relational.py` | `RelationalMemory` | 三元組讀寫 API |
| `index.py` | `MemoryIndex` | 輕量目錄（fact count / skill count / episode count）|
| `search.py` | `MemorySearch` | BM25 + cosine similarity 兩階段搜尋 |

### loom/core/cognition/（LLM 調用管理）

| 檔案 | 類別 / 函數 | 說明 |
|------|-------------|------|
| `providers.py` | `LLMProvider`, `MiniMaxProvider`, `AnthropicProvider` | Provider 抽象 + 具體實現 |
| `router.py` | `LLMRouter`, `stream_chat` | 前綴路由 + async streaming |
| `context.py` | `ContextBudget` | Token 配額追蹤 + `_smart_compact()` |
| `reflection.py` | `ReflectionAPI` | session_summary / tool_success_rate / skill_health |
| `prompt_stack.py` | `PromptStack` | SOUL → Agent → Personality 三層注入 |
| `embeddings.py` | `MiniMaxEmbeddingProvider`, `cosine_similarity` | 向量嵌入（Phase 5）|

### loom/core/tasks/（DAG 任務引擎）

| 檔案 | 類別 / 函數 | 說明 |
|------|-------------|------|
| `graph.py` | `TaskNode`, `TaskGraph` | DAG 節點 + Kahn's topological sort |
| `scheduler.py` | `TaskScheduler` | asyncio.gather 同層並行 + stop_on_failure |
